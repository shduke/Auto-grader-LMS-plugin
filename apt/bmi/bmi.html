APT: BMI

Specification

filename: BMI.py def calculate(weight,height): """ return float indicating BMI (body mass index) given weight in pounds (float) given height in inches (float) """ # you write code here

Problem Statement

See Wikipedia for information about body mass index or BMI and how to calculate it using the formula 

 BMI = 703.0695 * weight/(height2)

Write function calculate that returns the BMI of a person whose weight and height, in pounds and inches respectively, are parameters to the function calculate.

Constraints

  • weight will be greater than zero

  • height will be greater than zero

Examples

  1. weight = 200
height = 60

returns 39.059


  2. weight = 170
height = 72

returns 23.056


  3. weight = 53
height = 250

returns 62.573



  4. weight = 70
height = 120

returns 17.218
